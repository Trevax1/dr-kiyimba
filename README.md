## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm (v6 or higher)
- Modern web browser

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Trevax1/dr-kiyimba.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file with required environment variables
4. Start the development server:
   ```bash
   npm run dev
   ```
